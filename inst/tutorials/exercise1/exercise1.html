Package Install, revisited

While we have learned about R packages already, section 1.3.1 is a useful refresher, and also introduces an easier installation procedure available in RStudio.

Note: if you do not have the “Packages” tab in the “Files” pane as described, try the following:

  1. Go to Tools –> Global Options…
  2. Select the Packages tab on the left
  3. Ensure that “Enable packages pane” is checked
  4. Select Pane Layout on the left
  5. In the bottom right pane, make sure that “Packages” is selected
